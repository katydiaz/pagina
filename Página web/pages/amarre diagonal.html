Amarre Diagonal


Para qué sirve:



El amarre diagonal se emplea cuando dos troncos se cruzan en un ángulo que no es de 90 grados. Este amarre proporciona una mayor estabilidad a la estructura, especialmente cuando los troncos se cruzan en ángulos agudos u obtusos. Es ideal para reforzar estructuras y evitar que se desplacen lateralmente. Se usa comúnmente en la construcción de trípodes, refuerzos de esquinas y en las bases de torres, donde se necesita un soporte adicional para mantener la estructura firme.



Paso a paso:



1. Se hace un nudo de leñador alrededor de los dos troncos cruzados.

2. Dar por lo menos tres vueltas siguiendo la dirección del nudo leñador.

3. Se dan tres vueltas de forma cruzada a las vueltas anteriores.

4. Ahorcar el amarre dando por lo menos tres vueltas de forma cruzada de las vueltas anteriores.

5. Termina el amarre con un ballestrinque.
